Linux: можливі проблеми з доступом до серійних портів

На деяких дистрибутивах Linux порти можуть не відображатись або бути недоступні, якщо поточний користувач не входить до групи dialout. Щоб надати доступ, виконайте в терміналі:

sudo usermod -a -G dialout $USER

Після цього потрібно вийти із сеансу та увійти знову (або перезавантажити систему).  Докладніше про роботу з серійними портами в Linux →

Web Serial API не підтримується

Для роботи з серійними портами потрібен браузер з підтримкою Web Serial API.

Google Chrome 89+  або  Microsoft Edge 89+

Safari та Firefox наразі не підтримують Web Serial API.
